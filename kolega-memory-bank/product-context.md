# Product Context

## Why This Project Exists
This is a full-stack application template/project that provides a foundation for building modern web applications with authentication, data visualization, and API-driven architecture.

## Problems It Sololves
- Boilerplate reduction for React + TypeScript + Tailwind projects
- Pre-configured backend with database connectivity
- Ready-to-use UI component library (shadcn/ui)
- Authentication scaffolding

## How It Should Work
- Frontend runs on Vite dev server (default port 5173)
- Backend runs on Hono server (default port 9002)
- Frontend proxies API requests to backend
- MongoDB used for persistent data storage

## User Experience Goals
- Fast, responsive UI with modern design patterns
- Secure authentication flow
- Clear navigation and data presentation
