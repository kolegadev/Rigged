# Project Brief

## Overview
Full-stack web application built with React + Vite + shadcn/ui frontend and Hono + MongoDB backend.

## Core Requirements
- Modern React frontend with TypeScript
- Hono-based backend API with MongoDB persistence
- Authentication and user management
- Markets/data visualization features

## Goals
- Provide a robust full-stack application template
- Support scalable development with clear separation of concerns
- Enable rapid feature development through established patterns
